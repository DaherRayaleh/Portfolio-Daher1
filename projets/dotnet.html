GSB Mécénats

Language utilisé: C#, SQL
Architecture applicative: MVC (Modèle-Vue-Contrôleur)
Environnement technologique: Visual Studio Code
Gestion de projet: Projet d'équipe

Contexte

Il s'agit de la gestion des mécénats pour l'amélioration du site Galaxy Swiss Bourdin (GSB), une entreprise fictive.
Nous corrigeons les problèmes existants et ajoutons de nouvelles fonctionnalités en développant des méthodes dans les modèles, les vues et les contrôleurs (MVC).
Notamment pour la gestion CRUD et l'ajout de données utilisateurs.
